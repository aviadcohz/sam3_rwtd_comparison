About AutoSAM

AutoSAM replaces SAM's manual prompt encoder with a learned image encoder (HarDNet backbone). It uses only the image — no text descriptions or point prompts. SAM's weights are frozen; only the small ModelEmb (~2M params) is trained.

Unlike all other approaches which are zero-shot, AutoSAM is trained on 26 RWTD training samples. It outputs a single binary mask; the second texture is the complement (1 − mask). Hungarian matching selects the best label assignment during evaluation.
